# 🐦 Quiz Ornithologique — France & Outre-mer

Quiz interactif d'identification des oiseaux de France métropolitaine et d'outre-mer, avec photos tirées de Wikimedia Commons.

## 🌍 Territoires couverts

| Territoire | Espèces | Sources |
|---|---|---|
| 🇫🇷 France | 367 | LPO, oiseaux.net |
| 🏝️ La Réunion | 66 | SEOR |
| 🌺 Mayotte | 54 | GEPOMAY |
| 🌴 Antilles (Guadeloupe & Martinique) | 151 | Amazona, SOSDOM |
| 🌳 Guyane | 185 | GEPOG |

**Total : 737 espèces**

## ✨ Fonctionnalités

- **Photos dynamiques** depuis Wikimedia Commons (plusieurs photos par espèce)
- **Filtres multiples** : par famille taxonomique, habitat, territoire, statut, difficulté
- **Mode examen** avec tirage pondéré (2/3 communes, 1/3 rares)
- **Statuts détaillés** : endémique, indigène, introduit, migrateur — avec code couleur
- **Critères d'identification** et pièges de confusion affichés après chaque réponse
- **Layout adaptatif** : photo + réponses côte à côte sur desktop, empilé sur mobile
- **Filtrage intelligent des photos** : rejette dessins, cartes, spécimens de musée, mauvaises espèces

## 🚀 Utilisation

Fichier HTML unique, aucune installation requise. Ouvrir `index.html` dans un navigateur.

Nécessite une connexion internet (chargement des photos depuis Wikimedia).
